Project Proprosal: The Journey to Weight Lose

Proprosal: I want to create a project about my short film "The Journet to Weight Loss" about a young kid who is on a journey to lose weight. I want to have the audio of my film playing in the background on the webstie while users click on screenshots on each of the slide. Each screenshot will tell a different story of my film. I am going to use circle from the 2D Primitives shape on p5.Js reference page. I want to use the mouseX and mouseY command for my website so users can move the mouse with the circle. I'm going to have four pages, Homepage, custom names for each page to make it unique, each page will be called Journey 1, Journey 2, and Journey 3. My goal is to make this website as visually appealing as I can.
